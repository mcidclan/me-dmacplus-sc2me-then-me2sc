## Dmacplus sc2me Sample Code

Implementation of 'dmacplusInitLLIs', enabling the transfer of n data bytes from Sc to Me and vis versa by constructing the required LLIs. The example transfers a 0x40000-byte block from sc to me then me to sc.

# Requirements
Run `./build.sh` from a bash shell.
